# `@workspace/api-zod` — Generated Zod schemas

Zod request- and response-validation schemas for every Temple TV API
endpoint, plus matching TypeScript types. Generated by **Orval** from
[`@workspace/api-spec`](../api-spec/README.md).

---

## 1. Usage

### Server side (`@workspace/api-server`)

```ts
import { ImportVideoBody } from "@workspace/api-zod";

router.post("/admin/videos", async (req, res) => {
  const parsed = ImportVideoBody.safeParse(req.body);
  if (!parsed.success) {
    return res.status(400).json({ error: "invalid", issues: parsed.error.issues });
  }
  // parsed.data is now fully typed
});
```

### Client side (admin / TV / mobile)

```ts
import type { Video } from "@workspace/api-zod";
const v: Video = await fetchVideo(id);
```

```
    └── generated/           ← Orval output — DO NOT EDIT BY HAND
```

---

## 3. Regenerating

```bash
pnpm --filter @workspace/api-spec run codegen
```

This regenerates `src/generated/**` from `lib/api-spec/openapi.yaml`. Commit
the diff to keep the generated types in lockstep with the API spec.
