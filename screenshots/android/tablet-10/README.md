# Android 10-inch Tablet Screenshots

Place up to 8 landscape screenshots here at **1920 × 1200 px** (JPG or PNG, max 8 MB each).

## Recommended sequence

| Filename | Screen |
|---|---|
| `01-live-broadcast.png` | Full-screen live broadcast with sidebar chat/prayer panel |
| `02-sermon-library.png` | Sermon grid (more columns visible on wider screen) |
| `03-programme-guide.png` | Programme Guide — schedule view |
| `04-library.png` | Personal library / saved sermons list |

## Production

Use an Android tablet emulator (10.1" WXGA at 1280×800 scaled to 1920×1200)
or a physical Pixel Tablet. Landscape orientation recommended for all tablets.
