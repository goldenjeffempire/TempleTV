Enhance the broadcast management system to support direct video uploads into the Broadcast Queue from a local computer without requiring the video to first be manually added through the Video Library workflow.

Requirements:

* Add a direct “Add Video” upload feature within the Broadcast Queue interface.
* When the administrator clicks the “Add Video” button, a modern upload modal or interface should appear with:

  * Drag-and-drop upload support
  * File browser selection support
  * Multi-file upload capability (if applicable)
  * Upload progress indicators and status feedback
* Allow administrators to instantly upload local video files directly from their computer into the Broadcast Queue.
* Once upload is completed, the uploaded video should automatically:

  * Be processed and validated
  * Be stored correctly in the media storage system
  * Be saved to the database
  * Appear immediately in the Broadcast Queue
  * Become available for instant broadcasting or scheduled playback
* Ensure uploaded videos automatically generate required metadata such as:

  * Video title
  * Duration
  * File format
  * Thumbnail preview
  * Upload timestamp
  * Broadcast-ready status
* Implement seamless integration between the upload system, media processing pipeline, database layer, and broadcast engine.
* Ensure the upload process supports large video files with optimized handling, resumable uploads (if applicable), and robust failure recovery mechanisms.
* Add comprehensive validation for supported video formats, corrupted files, duplicate uploads, and upload security protections.
* Ensure the interface is modern, responsive, intuitive, and optimized for desktop and administrative workflows.
* Implement real-time synchronization so newly uploaded videos instantly appear in the active broadcast queue without requiring manual page refresh.
* Optimize the system for enterprise-grade scalability, performance, reliability, and production-level broadcasting operations.

The final implementation should provide a seamless and professional broadcast workflow where administrators can instantly upload videos directly from their computer into the Broadcast Queue and immediately use them for live broadcasting operations.
