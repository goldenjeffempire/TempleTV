Deploy the Temple TV app to Amazon Fire TV using the Android TV architecture.

1️⃣ Create Amazon Developer Account

Go to:

Amazon Appstore

Create:

Developer account
Company profile
Tax/payment setup
2️⃣ Prepare Fire TV-Compatible App

Your app must support:

D-pad navigation
Remote controls
Focus-based UI
Landscape layouts
TV-safe spacing
3️⃣ Configure Android Manifest

Add Fire TV / TV launcher support:

<uses-feature
    android:name="android.software.leanback"
    android:required="false" />

Inside activity:

<intent-filter>
    <action android:name="android.intent.action.MAIN"/>
    <category android:name="android.intent.category.LEANBACK_LAUNCHER"/>
</intent-filter>
4️⃣ Optimize Video Playback

Must support:

HLS streaming
adaptive bitrate
fullscreen playback

Recommended:

ExoPlayer
HLS .m3u8
5️⃣ Generate Production APK

Build release:

./gradlew assembleRelease

Output:

app-release.apk
6️⃣ Test on Fire TV Device

Enable developer mode on Fire TV.

Install:

adb install app-release.apk

Test:

remote navigation
playback
fullscreen
buffering
authentication
7️⃣ Create Fire TV Store Assets

Required:

TV screenshots
banner image
app icon
splash screen
8️⃣ Upload to Amazon Developer Console

Inside Amazon Appstore:

Create app
Upload APK
Select Fire TV distribution
9️⃣ Configure App Metadata

Add:

description
categories
privacy policy
support URL
🔟 Submit for Review

Amazon reviews:

playback quality
remote support
stability
TV optimization
