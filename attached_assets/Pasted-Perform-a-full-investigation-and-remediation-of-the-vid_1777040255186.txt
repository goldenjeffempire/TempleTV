* Upload initiation and file handling layer
* Backend upload processing and validation services
* Media storage and file persistence system
* Database record creation and update logic
* Media indexing and retrieval mechanisms
* Video Library frontend data fetching and rendering logic
* Broadcast system media availability integration
* Cache layers and state synchronization processes

  * Properly saved to storage
  * Correctly registered in the database

  * Not being written to the database

  * Queue for broadcast

* Implement robust logging and diagnostics to trace upload-to-library propagation failures.
